Maze Problem
=============

http://rubylearning.com/blog/2009/12/27/rpcfn-mazes-5/

[![Code Climate](https://codeclimate.com/github/roguerb/maze.png)](https://codeclimate.com/github/roguerb/maze)


Usage
-----

    bin/solver <filename>
    or pipe the maze on stdin

Testing
-------

    bundle install

To run the tests:

    $ bundle exec rake
